# Sprint 5 Report (2/15/26 to 3/15/26)

## [YouTube link of Sprint 5 Video](https://www.youtube.com/watch?v=cFabowBw-w8)

## What's New (User Facing)
 * Admin Users can edit existing entries in the inventory table (UPC Code, Product, Net Weight, or Quantity)
 * Admin Users can select a timeframe to generate reports from in the reports screen. Current inventory statistics will still be based on up-to-date information, but information based on student checkouts will be filtered
 * Small instruction boxes were added to a few user screens to help users operate the website

## Work Summary (Developer Facing)
During Sprint 5, we focused on finishing some key features, preparing and deploying the system for our clients to test. We added some features that had been on our backlog for a while (e.g. the Edit function on the inventory table) and then deployed the system. We set up an Intel NUC PC provided by our client with a lightweight Linux OS and cloned the project onto it. We also developed a guide for Admin Users to operate the system. After this was set up, we successfully handed the machine off to our clients for them to test!

## Unfinished Work
We were not yet able to implement a couple features. First of all, we still plan to refactor our UI to make it more consistent, clean, and applicable to any accessibility standards. This was incomplete due to illness within the team that slowed our work. We also were not able to yet implement a system for student volunteers to log their hours within the website. This was not done as the completed features took more time than anticipated, leaving us with no time for this last issue.
